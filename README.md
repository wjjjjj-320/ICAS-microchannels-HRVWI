# ICAS Microchannels and HR-VWI Plaque Enhancement

This repository contains the code used for the manuscript:

*"Microchannels are Associated with Gadolinium Enhancement in Intracranial Atherosclerotic Plaque"*

## Overview

The code includes scripts for:
- Calculation of the normalized enhancement index (nEI) from high-resolution vessel wall MRI
- Organization and analysis of optical coherence tomography (OCT) plaque features
- Descriptive statistical analysis
- Univariable and multivariable linear regression models
